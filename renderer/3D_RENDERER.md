# 3D Chess Renderer


The 3D Chess Renderer is a complete overhaul of the 2D rendering system, providing an immersive 3D chess experience with:

- **3D Board Squares**: Full 8x8 chess board rendered in 3D with perspective projection
- **3D Piece Models**: Distinct geometric models for all 6 piece types (Pawn, Rook, Bishop, Knight, Queen, King)
- **Interactive Camera**: WASD movement and arrow key rotation controls
- **Mouse Interaction**: Ray casting for accurate square and piece detection
- **Compatibility**: Maintains the same interface as the 2D renderer for seamless integration
